##  My experience switching to linux

<Image src={image} />

### Introduction

I've been getting tired of windows and their infamous updates that always seem to come at the worst time, and I've been hearing a lot about linux and how it is the best operating system for developers.
So I decided to dual boot linux on my main computer and see how it goes.

### First thoughts

In windows, everything is predefined for you, but in linux I had to install everything from scratch, configure everything, and learn how to use the bash terminal.
As corny as it sounds but I feel good typing commands in the terminal, it makes me feel like a real developer.

### My favorite things about linux

The terminal is by far the greatest thing about linux,
```bash
sudo apt install steam
sudo apt install code
sudo apt install git
```
These three commands, ran from my terminal, installed steam, visual studio code, and git on my computer, it is so much faster than going to the website and downloading the installer.

### I Use Linux Mint 
The many positive remarks surrounding linux mint and how beginner friendly it is, made me install it and I have to say I am very happy with it. Surprisingly It has a windows-like feel to it.
